# Upload Order

Use this order if you upload the repository through the GitHub browser.

1. Create the repository `madrid-air-quality-analytics-pipeline`.
2. Upload the files and folders inside this package, not the ZIP itself.
3. Commit everything with the message: `Initial standalone portfolio upload`.

Upload these items:

```text
README.md
UPLOAD_ORDER.md
.gitignore
requirements.txt
main.ipynb
worker.py
assets/
presentation/
docs/
data/
```

Do not upload the raw METRAQ CSV to GitHub.

The dataset should be downloaded separately from Hugging Face and kept locally as:

```text
metraq_air_quality.csv
```
